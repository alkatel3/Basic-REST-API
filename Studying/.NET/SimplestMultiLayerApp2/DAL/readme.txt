классы DAL:
ExchangeRate (сериализуемый) - данные о курсе, подлежащие сохранению в хранилище данных
XMLDataProvider (реализация IDataProvider) - класс-провайдер для сохранения данных в файл с помощью XML-сериализации 
DataContext (конкретный IDataContext) - класс контекста получения и сохранения данных, используя заданный IDataProvider
DataSource (конкретный IDataSource) - класс-точка доступа к данным из BLL, 
представляет собой конфигурацию заданного контекста и заданного провайдера данных
