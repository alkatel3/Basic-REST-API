классы BLL:
CurrentRate - класс бизнес-сущности Курс
RateException - исключение бизнес-логики
CarrencyExchangeService - класс бизнес-логики для работы с курсом валют, использующий для связывания с источником данных конкретный IDataSource
ReadWriteService - класс бизнес-логики для чтения и записи значений, использующий для связывания с источником данных конкретный IDataSource
