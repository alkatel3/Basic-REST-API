добавление процедуры и др. объектов РБД в EF модель после  создания модели:

открываем окно Entity Data Model Browser (оно будет пустым при неактивном окне дизайнера модели), 
в выпадающем меню Model.Store командой Update Model from Database в окне указываем добавляемые объекты 
(в том числе и процедуры). 
На появившейся в Model.Store процедуре в контекстном меню выбираем Add Function Import, 
в открывшемся окне следуем инструкциям (задаем имя функции как метода объекта контекста, 
тип возвращаемого значения, получаем информацию о столбцах (кнопка) и если возвращается не скаляр, 
создаем новый тип (кнопка)). 
После этого описание процедуры войдет схему EF, 
а сама процедура будет доступна через инкапсулирующий ее метод объекта контекста
